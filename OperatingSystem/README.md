## 🔥 Key Interview Answers (Memorize These!):

### **1. Process vs Thread**
"A process is an independent program with its own memory, while a thread is a lightweight unit within a process that shares memory. Chrome uses processes for tabs (isolation) and threads within each tab for performance."

### **2. Deadlock 4 Conditions**
1. Mutual Exclusion
2. Hold and Wait
3. No Preemption
4. Circular Wait
*"All 4 must be true simultaneously for deadlock!"*

### **3. Thrashing**
"When a system spends more time swapping pages than executing processes because RAM is insufficient. Solution: Close apps or add more RAM."

### **4. Why Chrome Uses So Much RAM**
"Multi-process architecture - each tab is a separate process for security and stability. Trade-off: More memory but better isolation."
